## 介绍

基于mpmath&asteval，让AI能够进行简单的高精度数学运算

## 安装

### 全新安装
- 从链接安装 - 输入以下链接
``` text
https://github.com/NekoiMeiov/astrbot_plugin_mathitools
```
- 若是安装失败，可以尝试直接克隆源码：
```bash
# 克隆仓库到插件目录
cd /AstrBot/data/plugins
git clone https://github.com/NekoiMeiov/astrbot_plugin_mathitools

# 控制台重启AstrBot
```
